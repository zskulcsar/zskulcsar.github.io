# Launching the Blog

March 21, 2026

This is the first post in the MkDocs-based version of the site.

The goal is to keep the stack simple:

- Markdown for content
- MkDocs for generation
- GitHub Pages for hosting

That leaves very little machinery between writing and publishing.
